## Background

The recommendations presented here were developed through a synthesis of peer-reviewed evidence, field experience, and expert input.  
A key component of this work was a **questionnaire distributed at the UK Acoustics Network (UKAN+) Ecoacoustics Symposium** (Manchester Metropolitan University, June 15â€“16, 2022), which gathered input from 84 ecoacoustic practitioners. The responses informed the development of best-practice recommendations for ecoacoustic study design, recorder settings, and deployment strategies.
